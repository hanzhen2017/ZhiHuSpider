# ZhiHuSpider
自写爬虫爬取知乎问题及回答   

for python v 3.7


安装说明

一、安装scrapy

    1、安装wheel
        pip install wheel
    2、安装lxml
        https://pypi.python.org/pypi/lxml/4.1.0
    3、安装pyopenssl
        https://pypi.python.org/pypi/pyOpenSSL/17.5.0
    4、安装Twisted
        https://www.lfd.uci.edu/~gohlke/pythonlibs/
    5、安装pywin32
        https://sourceforge.net/projects/pywin32/files/
    6、安装scrapy
        pip install scrapy
        
二、安装其它依赖

    pip install qyquery
    
    pip install pymysql
